## 🎯 نحوه استفاده

### در کد TypeScript:
```typescript
import { ThemeService } from 'jalali-date-picker';

constructor(private themeService: ThemeService) {}

// تغییر تم
this.themeService.setTheme('scifi');

// تغییر به حالت تاریک
this.themeService.toggleDarkMode();

// دریافت تم فعلی
const currentTheme = this.themeService.getCurrentTheme();

// دریافت لیست تمام تم‌ها
const allThemes = this.themeService.getThemes();
```

### در Template:
```html
<jalali-date-picker
  [theme]="'scifi'">
</jalali-date-picker>
```

## 📱 پشتیبانی از Dark Mode

تمام تم‌ها از حالت تاریک/روشن پشتیبانی می‌کنند:

```typescript
// تغییر به حالت تاریک
this.themeService.toggleDarkMode();

// بررسی حالت فعلی
const isDark = this.themeService.getCurrentTheme().isDark;
```

---

## 💾 ذخیره‌سازی تم

تم انتخاب شده به صورت خودکار در localStorage ذخیره می‌شود:

```typescript
// تم به صورت خودکار ذخیره می‌شود
this.themeService.setTheme('scifi');

// بازیابی تم ذخیره شده
const savedTheme = this.themeService.getCurrentTheme();
```
