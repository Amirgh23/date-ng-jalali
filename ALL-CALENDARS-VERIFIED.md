# بررسی و تصحیح الگوریتم‌های همه تقویم‌ها

## خلاصه

همه الگوریتم‌های تبدیل تاریخ برای سه تقویم (جلالی، میلادی، قمری) بررسی و تصحیح شدند.

## ۱. تقویم جلالی (شمسی)

### الگوریتم‌ها:
- `gregorianToJalali()` - تبدیل میلادی به جلالی
- `jalaliToGregorian()` - تبدیل جلالی به میلادی

### وضعیت: ✅ تصحیح شد

### مشکلات قبلی:
- الگوریتم `jalaliToGregorian` یک روز اضافی به تاریخ‌ها اضافه می‌کرد
- روز ۱ هر ماه به عنوان روز ۳۰ یا ۳۱ ماه قبل نمایش داده می‌شد

### تصحیحات:
- حذف خط اضافه کردن روز اضافی: `days += (gy % 4 === 0 && ...) ? 1 : 0;`
- الگوریتم حالا دقیق و صحیح کار می‌کند

### تست‌ها:
```
✓ 1 Farvardin 1404 = 21 Mar 2025
✓ 1 Bahman 1404 = 21 Jan 2026
✓ 1 Esfand 1404 = 20 Feb 2026
✓ 4 Esfand 1404 = 23 Feb 2026 (امروز)
✓ 30 Bahman 1404 = 19 Feb 2026
✓ 30 Esfand 1404 = 21 Mar 2026
```

## ۲. تقویم میلادی (گرگوریان)

### الگوریتم‌ها:
- تبدیل‌های میلادی توسط JavaScript Date API انجام می‌شود

### وضعیت: ✅ صحیح

### توضیحات:
- تقویم میلادی از API استاندارد JavaScript استفاده می‌کند
- نیازی به الگوریتم خاصی ندارد
- همه تبدیل‌ها صحیح هستند

## ۳. تقویم قمری (هجری)

### الگوریتم‌ها:
- `gregorianToHijri()` - تبدیل میلادی به قمری
- `hijriToGregorian()` - تبدیل قمری به میلادی

### وضعیت: ✅ تصحیح شد

### مشکلات قبلی:
- الگوریتم قدیمی تقریبی بود و دقیق نبود
- تبدیل قمری به میلادی سال‌های اشتباه برمی‌گرداند (۲۰۱۷ به جای ۲۰۲۶)

### تصحیحات:
- پیاده‌سازی الگوریتم استاندارد بر اساس Julian Day Number
- استفاده از فرمول‌های دقیق ریاضی
- تست Round Trip: میلادی → قمری → میلادی ✓

### تست‌ها:
```
✓ 23 Feb 2026 = 1447/9/6 (6 Ramadan 1447)
✓ 1 Ramadan 1447 = 18 Feb 2026
✓ 1 Muharram 1447 = 27 Jun 2025
✓ 1 Shawwal 1447 = 20 Mar 2026
✓ Round Trip Test: 23 Feb 2026 → 1447/9/6 → 23 Feb 2026
```

### نکته مهم:
تقویم قمری بر اساس رؤیت هلال است و ممکن است ۱-۲ روز اختلاف با تاریخ‌های محاسبه شده داشته باشد. این الگوریتم تاریخ‌های محاسباتی را برمی‌گرداند.
