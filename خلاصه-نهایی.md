## 🔧 رفع مشکلات احتمالی

### اگر تقویم نمایش داده نمی‌شود:

1. **Hard Refresh مرورگر**
   ```
   Ctrl + Shift + R
   ```

2. **پاک کردن Cache و Rebuild**
   ```bash
   rmdir /s /q .angular
   rmdir /s /q dist
   ng build jalali-date-picker
   ng serve demo
   ```

3. **بررسی Console**
   - F12 را بزنید
   - تب Console را باز کنید
   - خطاها را بررسی کنید

### اگر استایل‌ها اعمال نمی‌شوند:

1. **بررسی CSS Variables**
   در Console مرورگر:
   ```javascript
   const root = getComputedStyle(document.documentElement);
   console.log(root.getPropertyValue('--primary-color'));
   ```

2. **بررسی فایل app.scss**
   مطمئن شوید که CSS Variables تعریف شده‌اند
