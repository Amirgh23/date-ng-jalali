## نحوه استفاده / How to Use

### مرحله 1: باز کردن تقویم
کلیک روی فیلد ورودی تاریخ یا آیکون تقویم 📅

### مرحله 2: یافتن دکمه زبان
دکمه زبان در گوشه بالا سمت راست پنل تقویم قرار دارد (در کنار دکمه تم 🎨)

### مرحله 3: تغییر زبان
کلیک روی دکمه زبان برای جابجایی بین فارسی و انگلیسی

## تغییرات رابط کاربری / UI Changes

### زمانی که زبان فارسی است:
- جهت متن: راست به چپ (RTL)
- نام روزها: ش، ی، د، س، چ، پ، ج
- نام ماه‌ها: فروردین، اردیبهشت، ...
- دکمه‌ها: "امروز"، "انتخاب تم"

### زمانی که زبان انگلیسی است:
- جهت متن: چپ به راست (LTR)
- نام روزها: S, S, M, T, W, T, F
- نام ماه‌ها: January, February, ...
- دکمه‌ها: "Today", "Select Theme"

## ویژگی‌های بصری / Visual Features

### رنگ‌ها (Colors)
- **پیش‌فرض**: پس‌زمینه روشن با حاشیه خاکستری
- **Hover**: پس‌زمینه بنفش با متن سفید
- **Focus**: حاشیه آبی برای دسترسی‌پذیری

### اندازه (Size)
- عرض: 36px
- ارتفاع: 36px
- شعاع گوشه: 6px

### انیمیشن (Animation)
- انتقال نرم 0.2 ثانیه برای تمام تغییرات
- تغییر رنگ هنگام hover
- تغییر مقیاس کوچک هنگام کلیک

## مثال کد / Code Example

```html
<!-- استفاده ساده -->
<jalali-date-picker
  [locale]="'fa'"
  [(ngModel)]="selectedDate">
</jalali-date-picker>

<!-- با رویداد تغییر زبان -->
<jalali-date-picker
  [locale]="currentLocale"
  (localeChange)="onLanguageChange($event)"
  [(ngModel)]="selectedDate">
</jalali-date-picker>
```

```typescript
export class MyComponent {
  currentLocale: 'fa' | 'en' = 'fa';
  selectedDate: Date = new Date();

  onLanguageChange(locale: 'fa' | 'en') {
    console.log('زبان تغییر کرد به:', locale);
    this.currentLocale = locale;
  }
}
```

## نکات مهم / Important Notes

1. **ذخیره خودکار**: زبان انتخابی به صورت خودکار در مرورگر ذخیره می‌شود
2. **تغییر فوری**: تمام متن‌ها بلافاصله به زبان جدید تغییر می‌کنند
3. **سازگاری با تم**: دکمه زبان با تم انتخابی سازگار است
4. **دسترسی‌پذیری**: کاملاً قابل استفاده با کیبورد و صفحه‌خوان
